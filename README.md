## ✅ Todo List

Aplicação simples de lista de tarefas desenvolvida para praticar conceitos fundamentais de JavaScript, manipulação do DOM e persistência de dados no navegador.

## 🚀 Funcionalidades

➕ Adicionar tarefas

❌ Remover tarefas

🧹 Limpar lista

💾 Salvamento automático com LocalStorage


## 🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript(Vanilla)

LocalStorage


## 📚 Apredizados com o projeto

Persistência de dados no navegador

Manipulação do DOM

Eventos em JavaScript

Organização de código
